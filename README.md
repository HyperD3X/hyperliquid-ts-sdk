Includes:

- Examples for NodeJS apps
- Examples for ReactJS apps

## [SDK documentation](./packages/sdk/README.md)

## Examples
CLI example for NodeJS dApps:
```
nvm exec npm i
nvm exec npm run playground-cli
```

ReactJS example:
```
nvm exec npm i
nvm exec npm run playground-reactjs
```
